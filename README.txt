# UnityDemonstration
A C# coding project demonstrating various functions in Unity

These scripts display a variety of C# and Unity features that were implemented in the Unity Demo I created.
The presented features are all acieved through C# scripting including:

Moving Game Objects via Mathematical Models and Vector Calculations
Dynamically Augmenting Physics Behaviors
Quaternion Rotation of Game Objects
Object Interaction Through Tags, Layers, and Scripts
Creating/Destroying Game Objects
Dynamically Changing Materials
Turning On/Off Particle Systems
Triggering Animations

A full description of the project as well as a video demonstration of the project is viewable at:
https://oliverthompsongamedesigner.wordpress.com
